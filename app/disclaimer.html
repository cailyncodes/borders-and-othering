Disclaimer

The geographic boundaries displayed are not exact for several reasons. One, official boundary data for many countries are classified military data not accessible to the public. Two, even in instance where the public can access the data, borders are political features that can and often lead to disagreement in their precise measurement. Three, in order to load the data onto this website, it needed to be compressed, which causes some data loss. This imprecision does not prevent the use of this data as instructional and informative. However, no warranty nor sufficiency for a specific purpose is afforded. Please see the detailed disclaimer under the data tab for more information.
